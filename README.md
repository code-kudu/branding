# CodeKudu Branding

Official brand assets for CodeKudu. This repository contains logos and icons in multiple formats and sizes for use across CodeKudu products, websites, and marketing materials.

## Assets

- **Logo** — Full CodeKudu wordmark with logomark, available in default and dark variants.
- **Icon** — Standalone logomark, available in default and dark variants.

Each variant is provided in the following formats:


| Format | Use Case                                      |
| ------ | --------------------------------------------- |
| `.svg` | Web, scalable graphics                        |
| `.png` | General-purpose raster (1x, 0.5x, 2x, 3x, 4x) |
| `.pdf` | Print and documents                           |
| `.ai`  | Adobe Illustrator source                      |
| `.eps` | Legacy print workflows                        |
